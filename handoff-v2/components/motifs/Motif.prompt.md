The Pearloom line-ornament set — fourteen single-stroke editorial glyphs in one consistent hand (1.75px round caps), spanning every occasion. Use for section openers, list bullets, empty-state centerpieces, and sticker-pack decor.

```jsx
<Motif name="laurel" size={56} />
<Motif name="rings" color="var(--pl-terra)" />
<Motif name="dove" size={40} />
```

Names: `sprig laurel bloom rings dove candle star sun wave cake vine cresset arch feather`. Weddings lean `rings/laurel/bloom`, memorials `dove/candle/cresset`, milestones `cake/star/sun`. Override `color`/`accent` to match a theme. Import `MOTIF_NAMES` to build a picker.
