The Pearloom action control — pill-shaped, Geist body type, verb-first labels (lowercase except the first letter: "Begin a thread", never "GET STARTED").

```jsx
<Button variant="pearl" size="lg" onClick={start}>Begin a thread <Pearl size={8} /></Button>
<Button variant="ink">Press publish</Button>
<Button variant="ghost">Cancel</Button>
```

- `pearl` — iridescent gold surface; exactly ONE per surface (the primary CTA).
- `ink` / `olive` / `terra` — solid fills. `paper` — light card-fill with hairline. `ghost` — transparent with olive hairline, for secondary actions.
- Pairs naturally with `Pearl` as a trailing glyph. All variants lift + spring on hover/press.
