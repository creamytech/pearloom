The couple / host monogram — two initials set in Fraunces inside an editorial frame. The wedding keystone: save-the-dates, a wax-seal favicon, a footer sign-off, the loading splash.

```jsx
<Monogram left="M" right="J" frame="ring" />
<Monogram left="A" single frame="crest" />     {/* one host */}
<Monogram left="M" right="J" frame="interlock" accent="var(--pl-olive)" />
```

Frames: `plain` (letters only), `ring`, `crest`, `wreath`, `diamond`, `interlock`. The pearl sits at the frame's crown. Override `ink` / `accent` to fit a site's theme (e.g. olive frame on a garden theme). Letters use the real Fraunces webfont, so the monogram themes and scales cleanly to a favicon.
